Icons Directory for YT-DLP Studio
==================================

This directory should contain application icons.

Recommended icon files:
- app_icon.ico (Windows icon, 256x256)
- app_icon.png (General purpose, 512x512)
- download.png (Download button icon, 32x32)
- settings.png (Settings icon, 32x32)
- folder.png (Folder browser icon, 32x32)

Creating Icons:
--------------
You can create icons using:
1. Free online tools like: https://favicon.io or https://www.flaticon.com
2. Design tools like: GIMP, Inkscape, or Figma
3. AI tools like: DALL-E, Midjourney (for unique designs)

Icon Style Guidelines:
---------------------
- Modern, clean, minimalist design
- Solid colors or subtle gradients
- Clear and recognizable at small sizes
- Consistent style across all icons

Theme Suggestions for YT-DLP Studio:
-----------------------------------
- Colors: Blues (#0e639c), whites, grays
- Style: Flat, material design, or minimal 3D
- Symbols: Download arrow, play button, video frame
